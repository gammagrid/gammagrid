# Options Flow Tracker

Локальный дашборд для ежедневного (по требованию) сбора данных опционных
цепочек по watchlist тикеров и расчёта аналитических метрик позиционирования:
Put/Call Ratio, Max Pain, приближённый GEX, unusual activity, IV (текущий срез,
volume-weighted история, drill-down по контракту), day-over-day OI delta.

Полная спецификация — в [docs/TZ.md](docs/TZ.md).

## Требования

- Docker и Docker Compose

## Запуск

```bash
docker-compose up
```

Дашборд будет доступен на [http://localhost:8501](http://localhost:8501).

Данные хранятся в SQLite-файле `data/options.db` на хосте (volume) — переживают
перезапуск и пересборку контейнера.

## Использование

1. В сайдбаре слева введите тикер (например, `AAPL`) и нажмите **«Добавить»** —
   он появится в watchlist.
2. Нажмите **«Собрать данные»** — приложение получит текущую опционную цепочку
   по каждому тикеру watchlist через Yahoo Finance и сохранит снэпшот. Сбор
   ручной, автоматического расписания нет.
3. Выберите тикер в выпадающем списке над вкладками — откроются метрики:
   - **Обзор** — Put/Call Ratio во времени
   - **Max Pain / GEX** — max pain и приближённый профиль gamma exposure по выбранной экспирации
   - **Волатильность (IV)** — усреднённый IV по тикеру во времени, срез skew по цепочке, история IV конкретного strike/expiry
   - **Unusual Activity** — контракты с аномальным объёмом на последнем снэпшоте
   - **OI Delta** — изменение open interest между двумя последними снэпшотами

Историю по большинству метрик (кроме Put/Call Ratio, IV-среднего и OI Delta)
имеет смысл ждать несколько дней сбора — часть графиков требует минимум двух
снэпшотов.

## Локальный запуск без Docker (для разработки)

```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
PYTHONPATH=. streamlit run app/dashboard.py
```

`PYTHONPATH=.` обязателен: Streamlit добавляет в `sys.path` директорию самого
скрипта (`app/`), а не корень проекта, и без этого падает `ModuleNotFoundError:
No module named 'app'` — импорты в `dashboard.py` (`from app import ...`)
рассчитаны на то, что корень проекта виден в `sys.path`. В Docker то же самое
решено через `ENV PYTHONPATH=/app` в `Dockerfile`.

## Ограничения источника данных

`yfinance` — неофициальная обёртка над Yahoo Finance, без SLA и официальной
поддержки. Возможны задержки данных (15–20 минут), нерегулярное обновление
open interest внутри дня и временные блокировки при частых запросах.
Приложение логирует сбои сбора (видно на дашборде после нажатия «Собрать
данные»), но не пытается обходить блокировки.
